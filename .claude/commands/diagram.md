# Diagram Generation

Create visual diagrams for: **$ARGUMENTS**

## Diagram Types

1. **Component Diagram**
   - Show component relationships
   - Highlight data flow
   - Identify interfaces

2. **Architecture Diagram**
   - System architecture overview
   - Layer structure
   - External dependencies

3. **Sequence Diagram**
   - Interaction flows
   - API call sequences
   - Event chains

4. **Class Diagram**
   - Class hierarchies
   - Inheritance trees
   - Relationships

## Output

Generate diagrams using:
- Mermaid syntax (preferred for markdown)
- ASCII art (for terminal display)
- PlantUML (for complex diagrams)

Include explanatory text describing key components and relationships.
