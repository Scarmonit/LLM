# Performance Benchmark

Analyze performance characteristics of: **$ARGUMENTS**

## Benchmarking Tasks

1. **Time Complexity Analysis**
   - Big-O notation for algorithms
   - Identify performance bottlenecks
   - Compare algorithm efficiency

2. **Resource Usage**
   - Memory allocation patterns
   - CPU-intensive operations
   - I/O operations

3. **Optimization Opportunities**
   - Suggest faster algorithms
   - Identify unnecessary computations
   - Caching opportunities
   - Database query optimization

4. **Performance Tests**
   - Generate benchmark test code
   - Suggest profiling strategies
   - Load testing recommendations

## Output

Provide:
- Performance metrics table
- Bottleneck identification
- Optimization recommendations with code examples
- Expected performance improvement estimates

Use `think harder` mode for comprehensive performance analysis.
